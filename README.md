# Laravel My Golf Range（Portfolio Version）

Laravel 10 を使用して構築した **ゴルフ練習場予約サイト** のポートフォリオ公開用リポジトリです。  
認証、施設情報登録・料金登録、予約状況一覧、予約登録、予約変更・キャンセルなどの基本機能を実装しています。  

## 公開URL : <a href="https://portfolio-sh0212.com/laravel_my_golf_range/" target="_blank">https://portfolio-sh0212.com/laravel_my_golf_range/</a>

---

## 🚀 特徴 / 実装機能

### 認証機能
- ユーザー / オーナー / 管理者（複数ガード）
- 認証機能は自作
- メール認証
- ユーザー認証は2段階認証となっていますが、メール送信は開発時の確認だけとしております（ダミー画面で代替）
- 登録済のダミーユーザーは以下の通りです(30名分のダミーデータを登録済)

  | ユーザー | メールアドレス | password |
  |----------|---------|---------|
  | ユーザー | test1@mail.com ～ test30@mail.com | pass123 | 
  | オーナー | ******* | ******* |  

  ※今回、管理者側は「初期設定の機能のみ」のため非公開とさせて戴きました
---

### 出展者（オーナー）機能
- 施設登録（ゴルフレンジ・オプション・料金　等の初期データ登録）
- 料金改定登録
- メンテナンス日登録

---

### ユーザー機能
- 予約状況一覧（タイムテーブルでの予約表示、日別検索）
- 予約登録
- 予約一覧・変更・キャンセル
- 天気情報（向こう5日分）

## 技術スタック

- **フレームワーク:** Laravel 10
- **フロントエンド:** Blade / Vite / Tailwind少し

- **API:** OpenWeatherMap
- **開発環境:** Docker + Sail（開発）
- **メール:** Mailtrap / Laravel Mail / Queue
- **その他:** 公開用清掃スクリプト（clean.sh）
